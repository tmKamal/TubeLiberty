TubeLiberty Privacy Policy

Last updated: March 1, 2026

TubeLiberty is a Chrome extension that helps users manage their YouTube screen time by setting daily watch time and shorts limits, tracking streaks, and providing a usage dashboard.

Data Collection and Storage

TubeLiberty collects and stores the following data locally on your device using Chrome's storage API (chrome.storage.local):

  • Daily watch time and shorts count
  • Configured time limits and preferences
  • Streak tracking data (consecutive days within limits)
  • Viewing history summary (last 7 days — time watched, shorts count, videos count)

All data is stored entirely on your device. No data is transmitted to any external server, third party, or analytics service.

Data Sharing

TubeLiberty does not share, sell, or transmit any user data to anyone. There are no external network requests, no analytics, and no tracking.

Data Retention

Data is stored locally as long as the extension is installed. Uninstalling the extension removes all stored data.

User Rights

You can clear all extension data at any time by:

  • Uninstalling and reinstalling the extension
  • Clearing extension data via chrome://extensions

Changes to This Policy

Any changes to this privacy policy will be posted on this page with an updated date.

Contact

For questions about this privacy policy, open an issue on the GitHub repository.
